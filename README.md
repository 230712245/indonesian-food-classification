# Indonesian Food Classification

A deep learning project for classifying Indonesian food images using multiple Convolutional Neural Network (CNN) architectures.

## Models
- Custom CNN
- AlexNet
- VGG16

## Features
- Image classification using deep learning
- CNN architecture comparison
- Transfer learning with VGG16
- Model evaluation and testing
- Streamlit deployment

## Files
- `custom_cnn_food_classification.ipynb` → Custom CNN training notebook
- `alexnet_food_classification.ipynb` → AlexNet training notebook
- `vgg16_food_classification.ipynb` → VGG16 training notebook
- `app.py` → Streamlit application
- `custom_cnn_model.h5` → saved trained CNN model
- `requirements.txt` → project dependencies

## Technologies
- Python
- TensorFlow / Keras
- OpenCV
- Streamlit
- Jupyter Notebook

## Description
This project was created to learn and compare deep learning models for Indonesian food image classification using CNN architectures and transfer learning techniques.
